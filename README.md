# ffmpeg-dashboard

A monorepo for a web dashboard to control a local instance of ffmpeg, built with Node.js, Fastify, and React/Vite.

Created as an educational project for first-year software engineering coursework at Kyiv Polytechnic Institute (KPI).

## Structure

- `client/`: React + Vite front-end app.
- `server/`: Node.js backend server (Fastify).
- `core/`: Shared core library (utilities, patterns).

## Quick Start

### Start both client and server

From the root directory:

```bash
npm install
npm run dev
```
